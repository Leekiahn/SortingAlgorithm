# 정렬 알고리즘 모음 (C#)

해당 저장소는 C# 정렬 알고리즘을 학습하면서 정리한 것입니다.
6가지 정렬 알고리즘은 32bit 정수형 타입의 배열을 인수로 받아 오름차순과 내림차순 정렬합니다.

## 알고리즘 리스트
- **버블 정렬 (Bubble Sort)**
- **삽입 정렬 (Insertion Sort)**
- **선택 정렬 (Selection Sort)**
- **퀵 정렬 (Quick Sort)**
- **병합 정렬 (Merge Sort)**
- **힙 정렬 (Heap Sort)**
